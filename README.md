# Calculator v1.0
    by Anirudh Jain
    Input types: Integer and floating point numbers
    Output types: Integer if all input are integers and the result is an integer. Or floating point number with precision 'double'
    Supported functions:
      1. Addition(+)
      2.Subtraction(-)
      3.Division(/)
      4.Multiplication(*)
      5.Mod(%)
